## 세션

Claude Code는 모든 대화를 세션 단위로 관리한다. 세션이란 Claude Code를 실행한 시점부터 종료할 때까지의 작업 흐름이다.

- 각 세션은 고유한 ID를 가진다.
- 세션 내 모든 대화가 컨텍스트로 유지된다.
- 세션을 종료해도 대화 기록은 자동 저장된다.
- 저장된 세션은 다시 열고 이어서 작업할 수 있다.

## 컨텍스트 관리

Claude Code는 사용자가 주고받은 모든 대화 내용과 파일 수정 기록, CLAUDE.md에 정의된 규칙 등을 하나의 컨텍스트로 유지한다. 컨텍스트는 Claude가 지금까지 이해한 모든 작업 기억이다.

컨텍스트는 무한하지 않기에 지정된 한도를 초과하면, 오래된 내용이 자동 축약되거나 제거된다.

### 컨텍스트 관리 명령어

- `/clear`: 세션의 모든 대화를 초기화
- `/compact`: 이전 대화를 스스로 요약해 컨텍스트 압축
- `/context`: 현재 세션의 컨텍스트 사용량 시각화
