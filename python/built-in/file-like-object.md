파이썬에서 파일과 같이 취급되어 사용할 수 있는 객체에 대해서 알아보자.

## io.BytesIO()

`io.BytesIO()` 는 메모리 영역에서 연결된 파일과 같은 객체를 생성한다. 이진 데이터(이미지, 오디오 등...)를 저장하는데 사용한다.

## io.StringIO()

`io.StringIO()` 는 메모리 영역에 연결된 파일과 유사한 객체를 생성한다. 텍스트 데이터(HTML, PHP 스크립트 등...)를 저장하는데 사용한다.

## tempfile.TemporaryFile()

`tempfile.TemporaryFile()` 는 메모리가 아닌 디스크에 임시 파일을 생성한다. 첫 번째 인수인 mode를 사용하여 플래그를 통해 파일에 이진 데이터를 저장할지, 텍스트만 저장할 지 선택할 수 있다.
